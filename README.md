## 📌 Project Overview

This project presents an interactive **Power BI Dashboard** for analyzing the performance of the **Mahatma Gandhi National Rural Employment Guarantee Act (MGNREGA)**. The dashboard provides insights into employment generation, expenditure, labour budget utilization, wage payments, and work execution, enabling data-driven decision-making for rural development.

---

## 🎯 Objectives

- Analyze MGNREGA expenditure and labour budget utilization.
- Monitor employment generation across districts.
- Evaluate worker participation and wage payments.
- Identify district-wise performance and work completion trends.
- Support policy decisions through interactive visualizations.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dataset

**Source:** Ministry of Rural Development, Government of India (MGNREGA)

The dataset contains information on:

- District
- Total Expenditure
- Approved Labour Budget
- Active Job Cards
- Active Workers
- Average Wage
- Average Employment Days
- Household Employment
- Completed Works
- Ongoing Works
- Wage Payment Percentage

## 📌 Key Insights

- Total expenditure reached **₹65.62 Million**.
- Labour budget approved was **₹16 Billion**.
- Average wage paid per worker was **₹6.73K**.
- Average employment generated was **16.21 days** per household.
- **97.75%** of wage payments were completed within the prescribed timeline.
- The dashboard helps identify high-performing and underperforming districts for better resource allocation.

---

## 🧠 SMART Analysis

### Descriptive Analysis
Summarizes employment generation, expenditure, labour budget utilization, and payment efficiency.

### Diagnostic Analysis
Identifies performance gaps by comparing expenditure, employment days, and budget utilization across districts.

### Predictive Analysis
Historical trends help forecast expenditure, labour demand, and employment generation.

### Prescriptive Analysis
Recommends improving labour budget utilization, increasing employment opportunities, and maintaining timely wage payments.
